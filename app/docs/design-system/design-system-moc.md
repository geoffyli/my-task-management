# Design System

A Linear-inspired design system supporting light and dark themes. Custom UI components built with Tailwind CSS 4 — no external component library. Theme preference (system/light/dark) is configurable per-browser via the Settings page.

The exhaustive visual spec lives in `/DESIGN.md` at the project root. These docs summarize the implementation details relevant to development.
